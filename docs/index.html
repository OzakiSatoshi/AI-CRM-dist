AI-CRM

名刺スキャンから一斉メールまで。あらゆるビジネスのための、Windows スタンドアロン CRM。

最新版をダウンロード 最新版を確認中…

対応: Windows 10 (1809) 以降 / 64bit ・ インストール時の管理者権限不要

主な機能

名刺スキャン (AI)

名刺・自己紹介シート・参加者リストを Claude が解析し、顧客情報を自動登録します。

顧客・案件管理

顧客、会社、タグ、セグメント、ランクを一元管理。重複の検出・統合にも対応。

一斉・選択メール送信

テンプレート・添付ファイル・地域絞り込みに対応した一斉／選択メール送信。

Google 連絡先連携

Google コンタクトと双方向同期。既存の連絡先資産をそのまま活用できます。

インポート / エクスポート

HubSpot CSV インポート、Excel／CSV 出力に対応。データの移行も容易です。

ローカル完結・安全

データは端末内の SQLite に保存。API キー等は Windows DPAPI で暗号化されます。
